# Changelog

Все значимые изменения в проекте будут документироваться в этом файле.

## [1.0.0] - 2024-12-20

### Добавлено
- Первоначальный релиз плагина
- Поддержка Email канала с HTML шаблонами
- Поддержка Telegram канала с MarkdownV2
- Система логирования в БД
- Админ-панель с настройками
- Статистика отправок
- Шорткод для тестирования
- Система хуков и фильтров
- Автоматическая очистка старых логов
- Поддержка мультисайтов
- Локализация (готовность к переводу)

### Технические особенности
- Объектно-ориентированная архитектура
- Singleton паттерн для основных классов
- Namespace готовность (можно легко добавить)
- Кэширование настроек
- Валидация всех входных данных
- Защита от XSS и SQL инъекций
- Поддержка Composer (структура)
